# DamuMed Digital Security Hub

This interactive dashboard is a practical MSc project artefact designed to mitigate 
privacy risks and improve cybersecurity awareness within Kazakhstan's e-health ecosystem.

## Project Timeline
- **Development Period:** January 2026 – May 2026
- **Final Release:** May 2026

## Deployment
The tool is accessible as a standalone Single Page Application (SPA). 
You can view the live version here: https://ayauzhanr.github.io/damumed-security-hub/
